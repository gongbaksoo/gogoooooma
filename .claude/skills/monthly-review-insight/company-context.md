# 회사 배경지식 (월 리뷰 고정 컨텍스트)

`/monthly-review-insight` 스킬이 **분석 전에 먼저 읽는** 고정 컨텍스트. 매달 다시 묻지 않아도 되는 회사·사업의 standing 사실을 여기 둔다.
- 표기: `[데이터 관찰]` = 매출 데이터에서 관찰된 것(시점 명시). `[확인 요망]` = 사용자가 채워야 할 전략·현장 사실(데이터로 단정 불가).
- 갱신: 사실이 바뀌면 이 파일을 직접 수정. 스킬은 항상 최신본을 읽는다.

---

## 1. 사업 개요
- 회사/팀: 이커머스팀 영업 (M&S 월간 리뷰 회의용 리뷰 작성).
- 취급: 영유아·생활용품 (브랜드 마이비/누비/쏭레브 등).  `[확인 요망]` 사업 한 줄 정의·핵심 카테고리.

## 2. 브랜드 구성
- 주요 3개 브랜드: **마이비 / 누비 / 쏭레브** (데이터 `품목그룹1` 기준).
- `[데이터 관찰 — 최근 12개월 매출 비중]` 마이비 ≈ 77% · 쏭레브 ≈ 13% · 누비 ≈ 10% → 마이비가 압도적 비중.
- 기타 브랜드(예: 데일리케어 등)도 일부 채널에 등장. `[확인 요망]` 3개 외 관리 대상 브랜드와 위상.
- `[확인 요망]` 각 브랜드의 포지션/타깃(예: 마이비=세탁·위생, 누비=수유·식기, 쏭레브=키즈 코스메틱?).

## 3. 채널 구성 & 분류
- 파트: **이커머스** vs **오프라인** (+ 전체).
- 이커머스 채널 예: 오픈마켓(사입), 오픈마켓(위탁), 자사몰, 종합몰, 버티컬커머스, 해외 등.
- 오프라인 채널 예: 할인점, 다이소, 대리점 등. (이마트·롯데마트 등은 '할인점'의 주요 거래처.)
- `[확인 요망]` "사입 vs 위탁"의 사업적 의미(마진·재고 책임 차이 등)와 자사몰의 전략적 위치.

## 4. 주력 채널 (전략)  `[확인 요망]`
- 회사가 **전략적으로 미는** 주력 채널은? (데이터 매출 1위 ≠ 전략 주력일 수 있음 — 예: 자사몰을 육성 중이라면 별도 강조.)
- `[데이터 관찰 — 최근 12개월 비중(일부 채널)]` 오픈마켓(사입) ≈ 64% · 자사몰 ≈ 15% · 오픈마켓(위탁) ≈ 13% · 할인점 ≈ 9%.
- 빈칸: 주력 ___ / 육성 중 ___ / 방어 ___.

## 5. 주력 상품 (브랜드별)  `[확인 요망]`
- `[데이터 관찰 — 2026-05 대상월 매출 상위, 백만원]`
  - 마이비: 얼룩제거제 223, 면봉 50, 순한라인 41, 삶기세제 30, 구강티슈 22.
  - 누비: 스텐 물병 17, 정글 물병 13, 롱핸들 4.
  - 쏭레브: 키즈 라인(키즈 썬케어/샴푸/페이셜클렌저) 중심.
- 빈칸: 회사가 정한 **핵심 캐시카우 / 전략 신상품 / 단종·축소 예정** 상품.

## 6. 주요 거래처  `[확인 요망]`
- 거래처 식별 기준 = 데이터 **R열(거래처명)**.
- `[확인 요망]` 매출·관계상 중요한 핵심 거래처(예: 이마트, 쿠팡 등)와 비고(전략 거래처/리스크 거래처).

## 7. 시즌성 · 주기 패턴  `[확인 요망]`
- `[확인 요망]` 계절 영향 큰 상품군(예: 쏭레브 키즈 썬케어=여름 성수기?), 명절·행사로 들쭉날쭉한 월, 월말 밀어내기 관행 등.
- 이게 있으면 "급감=부진"이 아니라 "시즌 종료=정상"으로 프레이밍 가능.

## 8. 목표 / KPI 기준
- 목표 파일: `full_targets_extracted.csv` (월·파트·(브랜드)별 목표). 달성률 = 실적 ÷ 목표.
- `[확인 요망]` 회의에서 가장 중시하는 지표(달성률 / 전년비 / 채널 믹스 / 특정 브랜드 등).

## 9. 단위 · 용어 규약
- 금액 단위: **백만원**(데이터 원값 ÷ 1,000,000, 반올림).
- 데이터 매핑: 브랜드 = `품목그룹1`, 거래처 = `R열(거래처명)`, 상품 = `품목 구분(S열)`, 채널 = `채널구분`.
- `대상 X` = 품목 미분류 placeholder → 분석에서 제외.

## 10. 알아둘 특이사항 / 주의  `[확인 요망]`
- `[확인 요망]` 일회성으로 크게 잡히는 이벤트(대량 딜·행사·반품), 데이터에 안 잡히는 외부 요인, 최근 조직/정책 변화 등 — 매달 헷갈리지 않게 적어둘 것.
