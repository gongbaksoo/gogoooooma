설계 결정 요약:
  • 섹션 위치 — 채널 종합 ↘ 바로 아래. 토글 OFF 시 통째로 숨김.
  • 그리드 = 3 컬럼 × 2 row (총 6 차트). 컬럼당 채널 그룹 1개 (사입/위탁/자사) — 그룹은 사용자 정의 가능 (P열 채널구분 매핑).
  • 거래처 차트 = R열 거래처명 기준 (영구 규약 §2.3.3.2). 브랜드 차트 = D열 브랜드 기준.
  • 각 차트는 자체 "표시 항목 수정" 모달 (ProductSelectionModal 재사용 패턴) — 옵션 풀은 해당 그룹의 row 필터 후 R/D열 unique.
  • part 토글(전체/이커머스/오프라인) 영향: 미정 — 18회차의 brand selection part 분기 패턴 따를지 사용자 확인 필요.
  • localStorage 저장 키: `avk_monthly_review_channel_issue_section` (그룹 정의 + 차트별 selection 통합).
  • "브랜드 상세" 섹션도 차트 표시 모달의 섹션 토글 대상으로 추가 (한 줄 토글로 통째 숨김).
⚠ 추가 확인 필요
  • part 토글(전체/이커머스/오프라인) 적용 여부 — 적용하면 selection도 part 분기 필요.
  • 그룹 default 정의 — 사입몰=`오픈마켓(사입)`, 위탁몰=`오픈마켓(위탁)+종합몰+버티컬커머스`, 자사몰=`자사몰` 으로 시작할지.
