B-1 — 그룹 내 실/점선 분리

검정 4단계 중 1·2 시리즈는 실선, 3·4 시리즈는 점선. 핑크 4단계도 동일 패턴. 각 hue 그룹 내에서 실/점이 추가 분리 역할.

B안 변형: B-1 (그룹 내 분리) B-2 (hue=패턴) B-3 (4차원) B-4 (8 슬롯) B-5 (14 슬롯) 원안: B 원안

12개월 매출 트렌드 — 8 시리즈

이 팔레트의 매핑 규약

검정 그룹(4단계): 1·2번 실선, 3·4번 점선 — 옅은 회색이 점선이 되면서 선 두께도 줄어 시각적 위계 자동.
핑크 그룹(4단계): 동일하게 1·2번 실선, 3·4번 점선.

장점: hue 4단계 명도 풀세트 보존. 패턴이 옅은 단계의 약한 식별성을 보완. "옅음 + 점선"이 자연스러운 보조 위계.

단점: 같은 시리즈 그룹 안에서 명도 + 패턴 2축이 동시에 바뀌어 멘탈 모델이 약간 복잡(시리즈명이 명확하면 문제 없음).
